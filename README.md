# Démarrons

Démarrer avec `npx create-probot-app my-first-app`.

* `basic-js`: Probot basique en JS qui commente une issue créée.
* `basic-ts`: idem, en TypeScript.
* `checks-js`: valide ("check") un commit.
* `deploy-js`: lance un déploiement via GitHub sur une PR.
* `git-data-js`: crée une PR avec un nouveau fichier sur un repository où est installé l'application.

# Découvrons

* `.env`: fichier de configuration.
* Service `https://smee.io/` pour faire le pont entre GitHub et une App locale.
